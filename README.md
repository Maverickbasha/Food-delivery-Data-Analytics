#  Zomato Restaurant Data Analysis

##  Project Overview

This project analyzes Zomato restaurant data to understand food trends, customer preferences, and pricing patterns. The analysis is done using Python in a Jupyter Notebook.

---

##  Objectives

* Identify popular food categories
* Analyze restaurant ratings
* Study cost distribution
* Understand online ordering trends

---

##  Tools & Technologies Used

* Python 
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

##  Dataset

* Source: Kaggle (Zomato Dataset)
* The dataset contains:

  * Restaurant name
  * Online order availability
  * Table booking option
  * Ratings
  * Votes
  * Cost for two people
  * Food category

---

##  Analysis Performed

### 1. Popular Food Types

* Identified most common restaurant categories
* Visualized using bar chart

### 2. Online Ordering Trends

* Compared restaurants offering online delivery vs not

### 3. Cost Distribution

* Cleaned and analyzed cost data
* Observed pricing patterns across restaurants

### 4. Rating Analysis

* Processed rating values
* Visualized rating distribution

---

##  Results & Insights

* Most restaurants belong to specific food categories (like Dining, Cafes, etc.)
* Majority of restaurants support online ordering
* Cost varies widely depending on restaurant type
* Ratings mostly fall between 3.5 to 4.5
